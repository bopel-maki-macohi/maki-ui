A UI library for Flixel. A replacement for `flixel-ui` that (hopefully) avoids the complexity of `haxeui-flixel`

## Open Source

This project is open source! (clearly...)

If you have a suggestion or there's something wrong, you can go to [the Issues tab](https://github.com/bopel-maki-macohi/maki-ui/issues).

If you have a suggestion that you have an implementation for, or maybe you're fixing something that went wrong, you can go to [the Pull Requests tab](https://github.com/bopel-maki-macohi/maki-ui/pulls).
